タイトル: CSS Custom Propertiesを用いたデザインシステムの構築と動的テーマ切り替えの学習
内容:
Intermediate HTML and CSS Courseにて、CSS Custom Properties（CSS変数）の仕様と実践を学習しました。
従来のプリプロセッサ（Sass等）の変数とは異なり、CSS変数はブラウザがレンダリングするタイミング（ランタイム）で値が評価されるため、DOM構造に応じたスコープ制御や、JavaScriptを用いた動的な値の書き換えが可能である点を深く理解しました。
特に:root疑似クラスでグローバルな変数を定義し、それを上書きすることで「ダークモード/ライトモード」のようなテーマ切り替えを低コストで実装できるアーキテクチャは、スケーラブルなプロダクト開発において非常に強力であると認識しました。今後は「Design Tokens」としての活用も意識して実装に取り入れます。


Title: Building a Design System and Dynamic Theme Switching Using CSS Custom Properties
Content:
In the Intermediate HTML and CSS Course, I learned about the specifications and application of CSS Custom Properties (CSS variables).
Unlike variables in traditional preprocessors (such as Sass), CSS variables are evaluated at runtime when the browser renders. This allows for scope control based on the DOM structure and dynamic value rewriting using JavaScript.
In particular, I realized that the architecture of defining global variables in the root pseudo-class and overriding them to implement theme switching like "dark mode/light mode" at low cost is extremely powerful for scalable product development. Going forward, I plan to incorporate them into my implementations, with an eye toward utilizing them as "Design Tokens."

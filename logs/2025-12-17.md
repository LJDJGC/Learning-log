## Learning Topic: CSS Layout & Functions

### Understanding Positioning Behavior
Referring to the following documents, I organized the `position` property's behavior to clarify how elements are separated from the layout flow.
- [MDN: Positioning](https://developer.mozilla.org/en-US/docs/Web/CSS/position)
- [CSS-Tricks: Absolute, Relative, Fixed Positioning](https://css-tricks.com/absolute-relative-fixed-positioining-how-do-they-differ/)

**Learned:**
- `relative` shifts the positioning while maintaining the normal document flow.
- `absolute` removes the element from the document flow and uses the nearest non-static parent element as the basis (the concept of a containing block).
- I had used it intuitively in previous web pages I created, but now that I understand the specifications, I can identify the cause of unintended layout issues.

I'm currently learning [CSS Functions](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Functions).
